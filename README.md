# Scientific Calculator Project

## Overview
This repository contains the code for a scientific calculator built in JavaScript. The calculator performs various mathematical operations including basic arithmetic, trigonometric functions, logarithms, and more. It is designed to be user-friendly and efficient.

## Features
- Basic arithmetic operations: Add, Subtract, Multiply, Divide
- Advanced functions: Trigonometric functions (sin, cos, tan), Logarithmic functions
- User interface: Simple and intuitive UI for improved user experience

## Installation
To use the calculator, clone the repository and open the index.html file in your browser:

```bash
git clone https://github.com/rohankokkarikr-svg/calculator.git
cd calculator
open index.html
```

## Usage
Simply enter the desired mathematical expression and press the equals button to get the result.
